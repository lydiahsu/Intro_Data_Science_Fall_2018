"All models are wrong.": This is a famous quote from the statistician George Box. Essentially it means that although nobody ever knows what the true model is (and the true model might even not be able to be expressed in any math form!), some models are still useful. Of course one can quantitatively define what "useful" means; e.g. minimum loss function, maximum prediction rates.
